# Summit AI Visibility Audit Tool v2

A Streamlit application for auditing website AI visibility, powered by Google Gemini 2.0 Flash.

## Setup

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Run the app

```bash
streamlit run app.py
```

### 3. Configure

In the sidebar:
- Enter your **Google Gemini API key** (get one at https://aistudio.google.com)
- Enter the **URL** you want to audit
- Click **Run Audit**

---

## Audit Dimensions

The tool audits 10 dimensions, weighted by AI search impact:

| Dimension | Weight | Description |
|---|---|---|
| Crawlability & Bot Access | 20% | Can AI crawlers access and render the page? |
| Structured Data / Schema | 18% | Schema.org JSON-LD quality and coverage |
| LLM Content Signals | 15% | E-E-A-T, content depth, factual density |
| Meta & SEO Signals | 12% | Title, meta description, canonical, OG tags |
| Heading Structure | 10% | H1–H6 semantic hierarchy |
| ARIA Implementation | 10% | Accessibility attributes aiding AI parsing |
| Link Quality | 8% | Internal/external link anchor text quality |
| Image Alt Text | 7% | Alt text coverage and quality |
| AI Search Health | 5% | llms.txt, AI bot directives in robots.txt |
| Duplicate Content & Tags | 5% | Canonical tags, duplicate titles |

Scoring: 1–2 = Critical (red), 3–5 = Needs Work (amber), 6–10 = Good (green)

---

## Exports

Three export formats are available after each audit:

1. **Word Document (.docx)** — Full Summit-branded audit report with executive summary, dimension scores, detailed findings per dimension, and a prioritised recommendations table. Designed to share directly with clients.

2. **PDF Summary** — Dashboard summary PDF for presentations. Includes overall score, dimension breakdown table, and executive summary.

3. **JSON Data** — Raw audit data for integration with other tools or CRMs.

## Notes

- The tool fetches pages using a bot user-agent string (`SummitAuditBot/2.0`)
- JavaScript-heavy pages may return lower crawlability scores (by design — this reflects AI crawler behaviour)
- Gemini API usage: approximately 10–15 API calls per full audit
- Rate limiting: 300ms delay between Gemini calls
